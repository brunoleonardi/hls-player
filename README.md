# HLS Player

Aplicação React simples demonstrando como reproduzir streams HLS usando a biblioteca [`react-hls-player`](https://www.npmjs.com/package/react-hls-player). A página exibe:

- Player responsivo com controle e possibilidade de atualizar o stream em tempo real.

## Pré-requisitos

- Node.js 18+ (ou versão compatível com o Create React App).
- npm instalado.

## Como executar

```bash
npm install
npm start
```

O comando `npm start` inicia o servidor de desenvolvimento em `http://localhost:3000`.
